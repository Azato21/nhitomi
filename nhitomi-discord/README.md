# nhitomi-discord

This is nhitomi [Discord](https://discord.com) frontend written in Typescript.

### Prerequisites

- [Node.js](https://nodejs.org/en/)
- [Java Runtime 8+](https://www.java.com/en/download/) (for [genclient](../genclient.md))

### Setup

```shell
# Install dependencies
$ npm i

# Run locally
$ npm start
```
